后端开发学习路线-JAVA

1. 计算机概论

  1. 课程：CS50(2019)

2. C++基础

  1. 课程：黑马
  2. 参考书籍：《C Primer Plus》
  3. 学完数据结构和算法进行进阶：《C Primer》

3. 计算机基础

1. 数据结构和算法（重点、优先）

  1. 课程：浙江大学陈越
  2. 参考书籍：《数据结构与算法分析》
  3. 力扣刷题 - 代码随想录

2. 操作系统

  1. 课程：
  2. 参考书籍：《深入理解计算机系统》

3. 计算机网络

  1. 课程：
  2. 参考书籍：《计算机网络：自顶向下方法》

4. 计算机组成原理（了解）

  1. 参考书籍：《计算机组成原理》 《深入理解计算机系统》

4. 深入学习

  1. C++深入：《Effective C++》&& 《STL源码剖析》
  2. MySQL：《MySQL必知必会》
  3. Linux:《鸟哥的Linux私房菜》
  4. 设计模式：《大话设计模式》

2. JAVA

  1. 课程：黑马
  2. SE、JVM、多线程

3. 相关开发工具

  1. Git
  2. Postman
  3. Intellij IDEA
  4. Navicat
  5. Maven

4. 基础框架组件

  1. Spring boot
  2. MyBatis

5. 数据库

  1. MySQL
  2. Redis

避坑：JSP、Tomcat原理和Servlet、Sprint(源码，事务，MVC)、Jetty原理

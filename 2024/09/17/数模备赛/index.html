一、数学建模初识

1. 数学建模基础

  1. 本质：模型简历与求解（掌握算法及其应用场景）、论文呈现（配图与排版）
  2. 常见题型
    1. 机理分析
    2. 优化类
    3. 评价类
    4. 数理统计模型（主要为预测类）

      新手推荐预测类和评价类

  3. 评奖标准（详见ppt）

    假设合理性、建模创造性、结果正确性、表述清晰性

    1. 摘要
    2. 问题分析（多加流程图）
    3. 模型建立与求解：必须有结果、图尽量多且高大上、模型对比
    4. 公式融入业务名词
  4. 组队与分工
    • 建模、编程、论文

2. SPSSPRO

  1. 可使用自定义算法diy数据处理/分析
  2. 可使用帮助中心快速上手
  3. 客户端可以使用notebook，很便捷

    包括数据处理、数据分析、绘图等功能

3. Python

  1. pandas
    方法 结果
    dtypes 类型
    head/tail(n=2) 首尾
    shape 大小（行列）
    index 索引
    columns 列名
    df[‘列名’] df[[‘’,’’]] 某/多列值
    describe 数据统计分析摘要
    info 数据简明摘要
    df.to_numpy DataFrameToNumpy
    T 转置

| df.sort_values(by=’’) | (从小到大)某列排序 |
| df.sort_values(by=’’,ascending=False) | 从大到小 |
| df[‘’].nuique | 查空值数据数量 |
| df[‘’].value_counts | 某列值统计数据 |
| sum,mean | |
| df[df[‘’]>10],df[df[‘’].str.startswith(‘’)] | 筛选 |
| df.iloc[20:26,0:2] | 切片 左闭右开(20-25行,0-1列) 如果以0开头，可以省略 负数为倒数 |
| df.loc[df[‘’].isin([‘’,’’]),[‘’,’’]] | 多重筛选 |
| df.groupby(‘’).mean() | 分组平均 |

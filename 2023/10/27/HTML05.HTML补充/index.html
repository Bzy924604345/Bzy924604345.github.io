3. 全局属性

  1. id
    • id不可重复
    • 作用：可以与表单控件关联，也可以CSS与JS配合
    • 以下HTML元素中不可以使用：html,head,mata,script,style,title
  2. class
    • 给标签指定类名，通过CSS给标签设置样式
  3. style
    • 设置CSS样式
  4. dir
    • 内容的方向，值：ltr,rtl
    • 以下HTML元素中不可以使用：html,head,mata,script,style,title
  5. title
    • 文字提示
  6. lang
    • 以下HTML元素中不可以使用：html,head,mata,script,style,title

全局属性列表：https://developer.mozilla.org/zh-CN/docs/Web/HTML/Global_attributes

1. 运算符

1.算数运算符

  • 加减乘除模

2. 赋值运算符

  1. =：将等号右边的值赋予给左边, 要求左边必须是一个容器
  2. 加值再赋给自己
    运算符 作用
    += 加法赋值
    -+ 减法赋值
    *= 乘法赋值
    /= 除法赋值
    %= 取余赋值

3. 自增/自减运算符

符号 作用 说明
++ 自增 变量自身的值加1，例如: x++
– 自减 变量自身的值减1，例如: x–
  • 有前置和后置

4. 比较运算符

运算符 作用
> 左边是否大于右边
< 左边是否小于右边
>= 左边是否大于或等于右边
<= 左边是否小于或等于右边
=== 左右两边是否类型和值都相等（重点）
== 左右两边值是否相等
!= 左右值不相等
!== 左右两边是否不全等
  • 字符串比较：按照ASCII码值；多个字符串挨个比较

5. 逻辑运算符

符号 名称 日常读法 特点 口诀
&& 逻辑与 并且 符号两边有一个假的结果为假 一假则假
|| 逻辑或 或者 符号两边有一个真的结果为真 一真则真
! 逻辑非 取反 true变false false变true 真变假，假变真
A B A && B A || B !A
false false false false true
false true false true true
true false false true false
true true true true false

6. 运算符优先级

  1. 逻辑运算符优先级： ！> && > ||
  2. 参考网站：https://www.w3school.com.cn/js/js_precedence.asp

2. 语句

1. 语句与表达式

  1. 表达式：因为表达式可被求值，所以它可以写在赋值语句的右侧。num = 3 + 4
  2. 语句：语句不一定有值，比如 alert()、for、console.log()和break 等

2. 分支语句

  1. if分支语句
  2. 三元运算符
  3. switch语句
    1. switch case语句一般用于等值判断, if适合于区间判断
    2. switch + case一般需要配合break关键字使用,没有break会造成case穿透

3. 循环语句

  1. while循环
  • break：中止整个循环，一般用于结果已经得到, 后续的循环不需要的时候可以使用
  • continue：中止本次循环，一般用于排除或者跳过某一个选项的时候
  • 无限循环：while(true)、for(;;)
  1. for 循环
